# SMS-Android-V1

> SMS ( School Management System - 대덕소프트웨어마이스터고등학교 학교 관리 시스템 )

<br/>

<br/>

## 기술스택
* 언어 : Kotlin
* 아키텍쳐 : Clean Architecture, MVVM
* 라이브러리 : RxJava, Koin, Navigation, Retrofit2, Android Jetpack libraries 

## 주요기능

1. 학사일정

   - 달별로 학사일정 확인
   - 최대 4개까지의 학사일정 표시

2. 시간표

   - 나이스 api를 사용한 시간표
   - 해당 요일 표시

3. 외출

   - 신청양식에 맞게 외출신청
   - 내 외출 내역 확인
   - 오늘의 외출증 확인
   - 외출시 유의 사항 확인

4. 공지사항
   - 어드민에서 올린 공지 확인
   - Editor.js kit 사용
   - 공지 검색
5. 마이페이지
   - 비밀번호 변경
   - 개발자 소개
